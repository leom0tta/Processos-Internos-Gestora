FUNDQUOTE — Classificação por AssetClass

Quando a Gorila retorna type: FUNDQUOTE, o campo security_type não é informativo. Nesses casos o sistema usa o assetClass da Gorila para classificar automaticamente. Este mapeamento é fixo — para sobrescrever um fundo específico, use a aba Por Nome de Ativo.

AssetClass (Gorila) Classe de Ativo (Suitability)
FIXED_INCOME Fundos de renda fixa ativa / crédito
MULTIMARKET Fundos multimercado / macro
STOCKS Renda variável (fundos, ações, FIIs, ETFs, exceto de renda fixa)
OFFSHORE Ativos no exterior
CRYPTO Criptoativos
CASH Saldo em Conta
CURRENCY Ativos no exterior
TANGIBLE Renda variável (fundos, ações, FIIs, ETFs, exceto de renda fixa)
